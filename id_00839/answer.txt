POINT:馬鹿正直に全てのワードに対してWebAPIから取得を行ったので時間がかかってしまいました。
一文字ごとにaもしくはbが挟まれる、aの場合は無視できて、bの場合は直前の文字列を繰り返すというのは、acceptとaddの結果をみてわかったのですが、与えられる文字列にbの繰り返しがある場合どのようになるのかは実際の結果（wobble=>waoabblaea）を見て確かめました。

なお、以下、取得に使ったRubyスクリプトです。

require 'net/http'
require 'uri'

open("words.txt").lines do |w|
  w.chomp!
  a = Net::HTTP.get URI.parse("http://try.undo.jp/spacetalky/api.cgi?input=#{w}")
  puts "#{w} = #{a}"
end

またdecode用の関数です。

def decoder src
  last = src.length - 1
  decoded = ""
  for i in 0..last do
    c = i.even? ? src[i] : (src[i] == 'b' ? src[i-1] : "")
    decoded += c
  end
  decoded
end

open("result.txt").lines do |l|

  puts results[1].chomp + ":" + results[0]
